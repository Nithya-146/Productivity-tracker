# Daily Productivity Tracker

## Description
This is a **Daily Productivity Tracker** built with **HTML, CSS, and JavaScript**.  
It helps you track your daily habits, water intake, study time, technical skill practice, communication improvement, reading, and other activities.  
The app provides **daily, weekly, and monthly analysis** with charts to visualize your progress.

---

## Features
- Track daily habits like:
  - Drinking water (4 liters/day)  
  - Wake up before 5 AM  
  - Reduce screen time  
  - Self-study (college & technical skills)  
  - Improve communication skills  
  - Reading books  
  - Learn something new for 30 minutes  
  - Spoken Hindi practice  
- Complete/skip tasks with a **reason note**  
- **Daily progress percentage**  
- **Weekly and monthly analysis**  
- **Charts** for visual tracking  
- Mobile-first responsive UI  
- Data saved using **localStorage**
